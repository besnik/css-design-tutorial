Flex basis is set to 100px for 4 items. In total 400px. But flex container has limited width of 300px, so flex will shrink the items because with 100px width each they won't fit in. Ratio is set as: 0.5 : 1: 0 : 1. This means that 2nd and 4th item will shrink equally, 1st item will shrink 2x slower than 2nd and 4th, 3rd item won't shrink at all!

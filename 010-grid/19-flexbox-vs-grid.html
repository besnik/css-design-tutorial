This example shows difference between CSS Grid and CSS Flexbox. Use CSS Grid for 2D layouts, use CSS Flexbox to align items in one dimension (vertical or horizonal). Note how attribute `align-self` on child elements work for both layouts. Same works also for other attributes, also on grid/flex component e.g. `gap`.

CSS Grid:

1 align-self:stretch
2 align-self:flex-start
3 align-items:flex-end
4 align-items:flex-end
5 align-items:flex-end

CSS Flexbox:

1 align-self:stretch
2 align-self:flex-start
3 align-items:flex-end
4 align-items:flex-end
5 align-items:flex-end

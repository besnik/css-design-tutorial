Description: remember, grid is just bunch of vertical and horizontal lines defining rows and columns. Each colum is separated by two lines: left and right. The same goes for rows. When we define columns and rows, grid automatically defines lines that separates columns and rows and gives them a number. We can make use of it and reference cells or whole area using those numbered lines. Then we can put a content into those areas.

Numbering starts from top left corner. In our example there are 4 vertical lines that defines 3 columns inside them - numbering goes from left to right like 1,2,3,4. And 3 horizonal lines that defines two rows - numbering goes like 1,2,3 from top to botom.

Start developer tools of your browser (usually you press F12 key) and inspect html code. Locate our div element with grid class and you will see small button `grid` next to it. Click it and you will see details of the grid including numbered lines.

This is very powerful technique that allows you do create and re-arrange unique layouts (based on viewports). Note we don't need any html wrappers - our html is clean as it should be.
